# Markeon

A web app for writing in Markdown and exporting it as a polished PDF.

Think of it as a combination of three things:
- Notion for the writing experience
- Canva for the visual style controls
- Microsoft Word for the layout precision

The name comes from the Eevee evolution family (the -eon suffix). Markeon transforms plain Markdown into something beautiful. The visual style is inspired by Umbreon - dark and moody with glowing amber-gold accents.

---

## The core idea

You open the app, write Markdown in an editor on the left, and see a live preview on the right that looks exactly like a printed page. You can pick from a set of beautiful themes that control fonts, colors, spacing - everything. When you're happy, you hit export and get a PDF that actually looks designed, not like something dumped out of a Markdown converter.

The whole thing runs in your browser. No account, no server, no internet required after the first load. Your files are stored in the browser itself.

---

## What makes it different

Most Markdown-to-PDF tools are either too basic (no styling) or too complicated (LaTeX, command line). Markeon sits in between - it gives you real design control through a visual interface, not config files or code.

The style inspector is the key feature: click any element in the preview, and controls for that specific element appear in a sidebar. Change the heading font, adjust the margin, tweak the color - all live, all instantly visible.

---

## Who it's for

Primarily for personal use. Students writing research papers, developers documenting projects, anyone who writes in Markdown and wants the output to look professional without fighting with a PDF library.

---

## What it supports

- Standard Markdown plus tables, task lists, footnotes
- Math equations (LaTeX syntax)
- Code blocks with syntax highlighting
- Diagrams drawn in text (Mermaid)
- Custom fonts (upload your own or pick from the built-in collection)
- Multiple themes with live preview
- Page layout controls: paper size, margins, orientation, headers and footers

---

## The goal

It should feel like a premium, polished tool. Not a side project. Not a prototype. Something you'd actually use every day and enjoy using.
